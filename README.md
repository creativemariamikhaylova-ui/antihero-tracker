## ◢ О проекте

Трекер для трёх независимых шкал:
- **+ ТВЁРДЫЙ ВХОД** — тренировки, упражнения, заметки
- **◌ НИКОТИН-ПЕРЕРЫВ** — просто клик
- **▵ ЖИДКИЙ ВАЙБ** — просто клик

Никаких эмодзи. Только квадраты, неон и статистика.

---

## ◢ Функции

| Блок | Что умеет |
|------|-----------|
| Календарь | три независимых календаря, дни недели, навигация по месяцам |
| Твёрдый вход | выбор упражнений (пресс, руки, подтягивания, ноги, кардио, своё) + заметка |
| Графики | совмещённый, упражнения, тепловая карта за год, сравнение месяцев |
| Достижения | серии, рекорды, очки, уровни |
| Данные | автосохранение в localStorage, экспорт/импорт JSON |

## ◢ Установка

### Вариант 1: Просто открыть в браузере
```bash
# скачай файл
curl -O https://raw.githubusercontent.com/YOUR_USERNAME/antihero-tracker/main/index.html

# открой в браузере
open index.html
```

## ◢ Вариант 2: На iPhone (как приложение)

Открыть index.html в Safari

Нажать «Поделиться» → «На экран «Домой»»

Готово — иконка на рабочем столе

## ◢  Вариант 3: Xcode WebView (для iOS-разработки)
Скопируй index.html в проект и используй WKWebView
